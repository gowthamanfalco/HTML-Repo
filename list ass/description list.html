HTML
The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser. It defines the meaning and structure of web content. It is often assisted by technologies such as Cascading Style Sheets and scripting languages such as JavaScript
HTML ELEMENTS
element is a collection of start tag, attributes, end tag, content between them
HTML TAG
HTML tags are like keywords which defines that how web browser will format and display the content. With the help of tags, a web browser can distinguish between an HTML content and a simple content.
ANGLE BRACKET
Angle brackets are commonly used to enclose a code of some type.
HTML ATTRIBUTE
HTML attributes are special words used inside the opening tag to control the element's behaviour.
HTML COMMENT
The comment tag is used to insert comments in the HTML code. It is a good practice of coding, so that coder and the reader can get help to understand the code. It is useful to understand steps of the complex code. The comment tag is helpful while the debugging of codes
HTML OBJECT ELEMENT
The tag defines a container for an external resource. The external resource can be a web page, a picture, a media player, or a plug-in application. To embed a picture, it is better to use the tag.
HTML SPECIAL CHARACTER
What are HTML special characte In HTML, special characters are typically those that can't be easily typed into a keyboard or may cause display issues if typed or pasted into a web page. If you plan to use any of the special characters on this page, you should use either the HTML entity name or the HTML entity number.
